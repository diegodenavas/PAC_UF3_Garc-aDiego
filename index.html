INSTRUCCIONES

- Para comenzar pulse la ranura para las monedas en la parte inferior derecha e introduzca el numero de monedas con las que quiere jugar.

- Para jugar pulse el boton tirar.

- Para ver las monedas disponibles pulse el boton salir.

- Cada tirada tiene un coste de una moneda.

- Esta es la lista de premios:

---- Una zanahoria: 1 moneda.

---- Dos zanahorias: 4 monedas.

---- Tres zanahorias: 10 monedas.

---- 2 hortalizas iguales: 2 monedas.

---- 2 hortalizas iguales y una zanahoria: 3 monedas.

---- Tres hortalizas iguales: 5 monedas.
